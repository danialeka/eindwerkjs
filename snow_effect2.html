Sneeuw Effect

WebSnowjq.Js is a jQuery plugin made using HTML5 and javascript

to create a virtual snow fall on a web page.

WebSnowjq.Js is compatible with all browsers supporting HTML5.

No effort is required to integrate WebSnowjq.js in a website.
